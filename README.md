# Student Performance Analysis (Excel & Power BI)

## Objective
Analyze student performance data to identify patterns in scores, gender distribution, and pass percentage.

## Dataset
- Source: Kaggle
- ~1000 student records
- Features: gender, parental education, scores (math, reading, writing)

## Tools Used
- Microsoft Excel (Data Cleaning, Transformation)
- Power BI (Dashboard & Visualization)

## Steps Performed
- Removed duplicates and handled missing values
- Standardized text formats using formulas
- Created calculated columns (Total, Average, Grade, Pass/Fail)
- Built Pivot Tables for analysis
- Designed interactive Power BI dashboard

## Key Insights
- Overall pass percentage is 97%
- Average score is around 67.75%
- Performance varies across gender and grade levels
- Higher grades show consistent performance trends
